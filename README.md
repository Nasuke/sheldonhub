# sheldonhub
基于Koa.js的社区系网站后端api

## 安装
* pnpm install安装所有依赖
* 在项目根目录下创建.env文件，并创建相应的环境变量(运行后可依据报错来创建)

## 功能
- [x] 用户注册
- [x] 用户登录
- [x] 登录验证
- [x] 发表动态
- [x] 获取单个动态
- [x] 获取多个动态
- [x] 修改动态
- [x] 删除动态
- [x] 给动态添加标签
- [x] 发表评论
- [x] 回复评论
- [x] 删除评论
- [x] 获取某个动态的评论
- [x] 创建标签
- [x] 获取标签
- [x] 上传头像
- [x] 获取头像
